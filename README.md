# 🚗 Gerenciado de Funilaria

Sistema web para gerenciamento de orçamentos de funilaria e pintura automotiva, desenvolvido para facilitar o atendimento ao cliente, emissão de orçamentos e controle dos serviços realizados.

## 📋 Funcionalidades

* Cadastro de clientes
* Cadastro de veículos
* Criação de orçamentos
* Geração automática de PDF
* Compartilhamento de orçamento
* Histórico de orçamentos
* Controle de status dos serviços
* Armazenamento em banco de dados Supabase
* Interface responsiva
* Instalação como aplicativo (PWA)

## 🚀 Acesso Online

Acesse a aplicação:

**https://gerenciado-de-funilaria.vercel.app**

## 📱 Instalação como App (PWA)

### Android

1. Abra o link da aplicação no Google Chrome.
2. Toque no menu ⋮.
3. Selecione **Instalar aplicativo**.
4. Confirme a instalação.

### iPhone (Safari)

1. Abra o aplicativo no Safari.
2. Toque em **Compartilhar**.
3. Selecione **Adicionar à Tela de Início**.

## 🎯 Objetivo

O objetivo deste projeto é fornecer uma solução simples e eficiente para oficinas de funilaria e pintura realizarem orçamentos profissionais, acompanharem clientes e organizarem seus serviços.
